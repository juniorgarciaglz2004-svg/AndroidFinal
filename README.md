## Tarea final de Android

El objetivo de este proyecto es la creación de una aplicación en Android, que sea un CRUD de una base de datos de coches.

Permite también la interacción con el navegador:
- ver el repositorio en Github
- En el listado de coches, para buscar al dar click sobre el nombre

También ofrece interacción con la aplicación de mapas para ver los concesionarios de coches en Zaragoza.

### Tecnología

Lenguaje: Kotlin

Base de datos: Sql lite usando SQLiteOpenHelper
